# Curated Raw Sources

Files here must be de-secreted, intentionally selected, and limited to small sources or excerpts that support durable wiki pages.

Use this area for reviewed snippets, notes, or source excerpts that are useful enough to keep beside the project but too raw to become the main wiki narrative.

Example filename: `2026-05-13-api-notes.md`.

Do not add secrets, credentials, auth tokens, private customer data, full logs, database exports, generated dumps, or bulk copied outputs.
